CI/CD is a method used in software development. It helps to build, test, and deliver code faster and more reliably.

CI (Continuous Integration):
Whenever a developer writes new code, it is automatically checked and added (integrated) with the existing code.
👉 This helps catch errors early.

CD (Continuous Delivery):
The code that passes all checks is automatically prepared for release.
👉 It’s ready to go live anytime with just one click.

CD (Continuous Deployment):
The code is not just prepared, but also automatically deployed to production (live) without manual steps.
👉 Changes reach users quickly and frequently.
